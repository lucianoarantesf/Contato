# Portfólio Profissional | Luciano Arantes Filho

Este repositório tem como objetivo centralizar e publicar meu portfólio profissional na web.

Nele apresento minha trajetória como desenvolvedor, principais habilidades técnicas, experiências profissionais, formação, certificações, formas de contato e projetos em destaque.

O portfólio foi pensado para funcionar como uma vitrine profissional digital, facilitando a apresentação do meu perfil, da minha experiência e dos trabalhos que representam melhor minha atuação em desenvolvimento de software.
